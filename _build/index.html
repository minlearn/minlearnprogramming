    minlearn¶

    a tech book for “minlearnprogramming” theories, and social learning theories, a script used for installing devdeskos (and otheros) online,with poential debug support, and a composed implentmented OS towards book theories, a allinone pve redist, this all formed the "minlearnprogramming" family.

    just type ` wget -qO- inst.sh | bash ` in your ssh to start. or click left demo to find out the basic live usage.

    Find out More

    This project is still in development, and there are definitely going to be some weird issues sometimes, but when you find something please open an new issue for it to get solved.
